Useful CMD comands

  • cd .. // exit the main directory
  • cd "directory" // enters to the directory in where is the project
  • dir // shows all the files in the directory
  • mkdir "folder name" // create a new folder, make sure that u are in the directory
  • type nul > "filenametext" // create a new file in the folder
  • del /Q "filename" // removes a file
  • del /F /Q "filename" // forced remove
  • rd /S /Q "folder name" // removes folders
  • ren "original file name" "new file name"// renames a file
  • move /Y "original folder name" "new folder name" // renames a folder

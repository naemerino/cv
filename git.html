Usefuly GIT commands

  • git init // initialize git creating a repository, make sure that you are in the project directory
  • git status // shows wich files are commited in the modify status
  • git add "file name" moves the file into the stage area
  • git rm --cached "file name" we remove the file from the stage area
  • git add . // moves all the project into the staging area
  • git commit -m "description" // move the files from commit to commited (master branch)
  • git log // shows all the commits that you did and when you did those changes
  • git log --oneline // show the commits that you did in one line with the id of the commit
  • git checkout "id commit"// shows the code of what you had in that commit
  • git checkout master // shows the code of what you had in the master branch
  • git revert "id commit" // create a new commit with the reverted changes
  • git reset "id commit" // removes all the commits did at that time
  • git reset "id commit" --hard // removes for ever the commit doing at that time
  • git pull // Download the changes of the differents branches

How to interact in bitbucket

  • Create an account in: Bitbucket
  • Create a repository
  • In the bitbucket page, clone your repository

After that, follow the steps in CMD

  1. make sure that in CMD you are in the correct path
  2. git clone "url of your cloned repository" // clones a project into you local machine
  3. In windows moves the files from the cloned folder into the project
  4. git add . // moves all the project into the staging area
  5. git status // shows wich files are commited in the modify status
  6. git commit -m "description" // move the files from commit to commited (master branch)
  7. git status // shows that you have everithing cloned
  8. git push origin master //push you project into yoy online repository
  9. git pull origin master //Download the changes of you repository
